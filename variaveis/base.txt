Variaveis e Constantes:
Um conceito muito importante para o aprendizado de lógica de programação são as VARIAVEIS: 
São basicamente espaçõs onde podemos armazear diversos tipos de dados e usa-los posteriormente ao decorrer do código fonte ou algoritmo. O bom das variaveis é que é possivel nomea-las, assim, basta usar o nome atribuido que ele trara o dado armazenado. Elas possuem varios tipo, e isso pode variar de linguagem pra linguagem, porém alguns tipos são universais, eles são os primitivos:

String: Tipagem utilizada para declarar variaveis com valores de texto, ou seja, uma sequencia de caracteres de texto.

Int: Serve para determinar quando uma variavel armazena um valor numérico inteiro.

Float/Double: Determina valores de ponto flutuante na variavel, numeros reais ou com '.', os chamados decimais.

Boolean: Tipo lógico, é usada apenas para dois tipos de variavel, verdadeiro (True) ou falso (False).
Null: valores vazios.

Existem linguagem tipadas e as não tipadas. Nas linguagens tipadas, é necessário informar o tipo da variavel na hora de declarar, ja as não tipadas fazem isso automaticamente de acordo com o valor atribuido. 


Alem das variaveis, existem tambem as CONSTANTES, elas possuem o mesmo conceito deu uma variavel, pois seu nome ja diz, ela é contante, ou seja,seu valor nunca é alterado ao longo de um algoritmo.

No mundo da programação é normal nomear diversas coisas ao deccorer do coódigo fonte, com isso, é bom saber de algumas boas práticas para um melhor funcionamento do algoritmo. Um deles é saber que ao nomear as coisas não se ultiliza espaços, para isso, alguns métodos foram criados, um deles muito conhecido é o "camelCase". Esse método consiste em ao invés de usar um espaço para separar duas palavras, basta inicar a segunda palavra com letra maiuscula, por exemplo: não use "tipo produto" , use "tipoProduto".
